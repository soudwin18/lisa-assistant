## Trigger Words & Workflows

### "แผนวันนี้"
1. คุยกับ Soudwin เพื่อ lock in schedule วันนั้น
2. สร้าง daily note ที่ `~/context/admin/daily-notes/YYYY/MM-Month/YYYY-MM-DD.md`
   - ใช้ template จาก `~/context/admin/daily-note-template.md`
   - ใส่ schedule table ที่ตกลงกัน
3. Running log และ reflection template รอให้ Soudwin กรอกระหว่างวัน/ปลายวัน

### Background Logging (ทุก session)
- ทุก session ให้เปิด daily note ของวันนั้น ถ้ายังไม่มีให้สร้างจาก template
- เมื่อเกิดเหตุการณ์สำคัญ ให้เพิ่ม timestamped entry เงียบๆ ใน Running Log
- Triggers: จบ task, เริ่มอะไรใหม่, ติดขัด, ตัดสินใจ, เปลี่ยน context
- Format: `HH:MM — [event]`
- ทำเงียบๆ ไม่ต้องบอก Soudwin
